## 📋 프로젝트 소개

8개의 질문으로 나의 소비 성향을 두 축으로 분석합니다.

- **축 1:** 충동구매 ↔ 계획구매
- **축 2:** 가성비 소비 ↔ 감성 소비

결과는 4가지 유형 중 하나로 나옵니다:

| 유형 | 설명 |
|------|------|
| 💼 알뜰살뜰 엑셀 인간 | 계획형 + 가성비형 |
| 🌷 철저한 낭만주의자 | 계획형 + 감성소비형 |
| 🏹 할인 사냥꾼 | 충동구매형 + 가성비형 |
| 🎨 기분파 예술가 | 충동구매형 + 감성소비형 |

## 🚀 실행 방법

### 1. 저장소 클론

```bash
git clone https://github.com/poodlecho/streamlit-quiz-project.git
cd streamlit-quiz-project
```

### 2. 가상환경 생성 (선택)

```bash
python -m venv venv
source venv/bin/activate      # macOS/Linux
venv\Scripts\activate         # Windows
```

### 3. 패키지 설치

```bash
pip install -r requirements.txt
```

### 4. 앱 실행

```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 이 자동으로 열립니다.

# 💰 쇼핑 성향 테스트

## 📋 Part 1: 계획형 vs 충동구매형

### Q. 쇼핑하러 가기 전, 당신의 모습은?
- **A (계획):** 살 물건을 미리 리스트로 적어둔다.
- **B (충동):** 일단 가서 구경하다가 꽂히는 걸 산다.

### Q. 사고 싶은 물건이 생겼을 때 결제 버튼을 누르는 속도는?
- **A (계획):** 장바구니에 넣어두고 며칠 더 고민해 본다.
- **B (충동):** '고민은 배송만 늦출 뿐' 바로 결제한다.

### Q. 갑자기 큰 지출(예: 가전제품 고장)이 생겼을 때 당신은?
- **A (계획):** 비상금 등 예비비 안에서 해결하려고 노력한다.
- **B (충동):** '어떻게든 되겠지' 일단 할부로 지른다.

### Q. 마음에 쏙 드는 옷을 발견했는데 사이즈가 딱 하나 남았다면?
- **A (계획):** 집에 있는 옷들과 매치가 잘 되는지, 이번 달 의류 예산 안인지 따져본다.
- **B (충동):** "누가 채가면 어떡해!" 고민은 배송만 늦출 뿐, 일단 결제하고 본다.

## 🎯 Part 2: 가성비형 vs 감성소비형

### Q. 식당을 고를 때 더 중요하게 생각하는 포인트는?
- **A (가성비):** 양이 푸짐하고 가격이 합리적인 곳.
- **B (감성):** 인테리어가 멋지고 플레이팅이 사진 찍기 좋은 곳.

### Q. 친구의 생일 선물을 고를 때 기준은?
- **A (가성비):** 할인 혜택이 좋고 구성이 알찬 실속 세트.
- **B (감성):** 브랜드 감성이 있고 포장이 고급스러운 아이템.

### Q. 자취방에 놓을 의자를 고를 때 나의 기준은?
- **A (가성비):** "엉덩이만 편하면 장땡!" 리뷰 많고 가격 저렴한 실속형 제품.
- **B (감성):** "방 분위기가 살아야지!" 조금 불편하거나 비싸더라도 디자인이 예쁜 제품.

### Q. 카페에서 메뉴를 고를 때 나는?
- **A (가성비):** "역시 아아(아메리카노)지!" 양 많고 가격 저렴하거나, 실패 없는 가성비 메뉴.
- **B (감성):** "이게 시그니처구나!" 비싸더라도 사진 찍기 예쁘고 독특한 비주얼의 음료.
